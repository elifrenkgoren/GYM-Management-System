# 🏋️ Gym Management System

## 📌 Project Description
Gym Management System is a web-based application developed using Java Spring Boot and MySQL.

This project helps manage gym members and their membership information through a simple and modern interface.

---

## 🚀 Features

- Add new members
- Update member information
- Delete members
- View members list
- Membership Type (Basic / Premium / VIP)
- Trainer Name
- Monthly Fee
- Responsive web interface with Bootstrap

---

## 🛠️ Technologies Used

- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL
- Thymeleaf
- HTML / CSS
- Bootstrap

---

## 💡 Object-Oriented Programming Principles

This project includes:

- Encapsulation
- Inheritance
- Polymorphism

Example:
- `Member extends Person`

---

## 📂 Project Structure

- `entity` → Model classes
- `controller` → Handles requests
- `service` → Business logic
- `repository` → Database operations
- `templates` → HTML pages

---

## ▶️ How to Run

1. Clone the repository
2. Open project in Eclipse / IntelliJ
3. Configure MySQL database
4. Run `GymManagementApplication.java`
5. Open browser:

http://localhost:8080/
